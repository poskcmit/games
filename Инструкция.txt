///////////////////////////////////////////////////////////
//    ИНСТРУКЦИЯ                                         //
//    по настройке клавиатуры и тестированию ArduCristal //
///////////////////////////////////////////////////////////
1. Электронный набор ArduCristal - это универсальное программируемое управляющее устройство с элементами 
ввода-вывода и отображения электронной информации, предназначенное для использования в проектах 
бытовой автоматизации, игровых модулях, учебном процессе и т.п.
----------------------------------------------------------------------------------------------------------
2. ArduCristal состоит из:
 1) микроконтроллерной платы Arduino Uno;
 2) LCD Keypad Shield = шильд с ЖКИ 16 х 2, пятикнопочной аналоговой клавиатурой и кнопкой сброса Arduino;
 3) корпуса, изготовленного на 3D принтере (верхняя и нижня крышки, блок колпачков для кнопок.
----------------------------------------------------------------------------------------------------------
3. ArduCristal обеспечивает:
 1) 7 цифровых входов-выходов;
 2) 5 аналоговых входов;
 3) интерфейсы передачи данных (на основе цифровых входов-выходов) UART, SPI, I2C;
 4) подключение внешних датчиков и модулей для создания проектов систем малой автоматизации.
----------------------------------------------------------------------------------------------------------
4. Программа (скетч) ArduCristal_test.ino предназначена для:
 1) точного определения и запоминания числовых констант каждой из кнопок (Select, Left, Down, Up, Right)
    аналоговой клавиатуры LCD Keypad Shield;
 2) отображения значений констант 5 кнопок аналоговой клавиатуры;
 3) установку яркости подсветки ЖКИ по 25 градациям и отображение их числовых констант;
 4) проверку формирования и отображения на экране ЖКИ пользовательских символов и простой анимации;
 5) просмотра вида и числовых констант символов таблицы встроенного знакогенератора.
----------------------------------------------------------------------------------------------------------
5. Порядок работы с программой ArduCristal_test.ino
 1) после запуска программы и отображения стартового текстового сообщения предлагается определить 
    константы всех кнопок аналоговой клавитуры:
    ------------------
    |16x02 LCD Keypad|
    |Define all keys!|
    ------------------
 2) константы кнопок определяются в последовательности Select(S)-Left(L)-Down(D)-Up(U) путем пятикратного 
    нажатия каждой кнопки после текстового запроса со счетчиком количества оставшихся нажатий и констант  
    каждого измерения:
    ------------------
    |Press S-key 5 tm| Пример экрана начала определения константы кноки Select
    |S-value =       |
    ------------------
 3) константа кнопки Right всегда имеет значение около 0, поэтому она устанавливается автоматически без 
    необходимости нажатий кнопки;
 4) усредненное значение всех констант измерений пяти нажатий каждой из 4-х кнопок временно отображаются 
    на ЖКИ, запоминаются программой для последующего просмотра, являются константами для данного шильда 
    LCD Keypad и могут в дальнейшем использоваться в прикладном программировании задач пользователя;
 5) после завершения определения констант всех кнопок программа предлагает меню проведения любого из 4-х   
    тестов после нажатия одной из четырех кнопок, показанных на экране в виде стрелок:
    Left  - Bright: установка яркости подсветки ЖКИ;
    Right - Char: проверка формирования и отображения на экране ЖКИ пользовательских символов;
    Up    - SymbTab: просмотр вида и числовых констант символов таблицы встроенного знакогенератора;
    Down  - Keys: отображение значений констант 5 кнопок аналоговой клавиатуры;
 6) выход в исходное меню из тестов Bright, Char и SymbTab производится нажатием кнопки Select,  
    а из Keys - автоматически через 4 секунды.
